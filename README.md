# ZernoFi

Real Yield from Ukrainian Agribusiness on Solana.

RWA-aggregator that lets crypto investors earn stable USDC yield from real agribusinesses (wheat producers, snail farms) through tokenized pools on Solana.

## Local development

```bash
npm install
npm start
```

Open http://localhost:3000

## Structure

- `public/index.html` — landing page
- `public/app.html` — investor dashboard
- `server.js` — Express static server

## Deploy

Auto-deploys to Railway on push to `main`.
